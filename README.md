# Wellwear

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 20.3.9.

'WellWear' es una web realizada por Guillermo Quintanilla Rodrigo para el Master Universitario de Aplicaciones y Sitios Webs, de la UOC.

Para ejecutar el proyecto, es necesario hacer antes 'npm install'

Para compilarlo y subirlo a producción, 'ng build'
